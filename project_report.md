# Yearly Agricultural Data Analytics and Business Intelligence System

## Company Name

Bindisa Agriculture Private Limited

## Abstract

This project presents a professional Yearly Agricultural Data Analytics and Business Intelligence System for Bindisa Agriculture Private Limited. The system analyzes agricultural production, crop yield, weather effect, cost distribution, revenue generation, and profitability across multiple Indian regions and crop categories. Python is used for data generation, data analysis, visualization, Excel reporting, and machine learning based yield prediction. MySQL is used as the central data warehouse where advanced SQL queries, views, aggregations, ranking logic, and year-over-year growth analysis are performed. Advanced Excel and Power BI are used to create professional dashboards for business decision-making.

## Objectives

- To generate a realistic yearly agriculture business dataset.
- To analyze crop-wise, region-wise, and year-wise performance.
- To identify revenue, cost, profit, and yield trends.
- To study the impact of rainfall and temperature on yield.
- To create SQL-based business intelligence views.
- To build an Excel dashboard using summaries and charts.
- To design a Power BI dashboard connected to SQL.
- To use machine learning for crop yield prediction.

## Problem Statement

Agriculture companies handle yearly data related to crop production, cultivation area, weather, operating cost, revenue, and profit. Without a proper analytics system, it becomes difficult to identify profitable crops, high-performing regions, cost inefficiencies, and weather-related yield patterns. This project solves the problem by creating an integrated data analytics and BI system using Python, SQL, Excel, and Power BI.

## Methodology

The project follows a complete analytics pipeline. First, Python generates a realistic agriculture dataset for Bindisa Agriculture Private Limited. The dataset is analyzed using Python libraries such as Pandas, Matplotlib, and Seaborn. Summary CSV files and chart images are created. MySQL stores the dataset as a structured data warehouse. Advanced SQL queries are used for KPIs, rankings, growth analysis, and BI-ready views. Excel creates a professional workbook dashboard, while Power BI connects to SQL views for interactive reporting. Finally, a Random Forest regression model predicts future crop yield.

## Dataset Description

The dataset contains yearly agricultural records from 2018 to 2026. It includes region, crop, season, area, production, yield, rainfall, temperature, cost categories, revenue, profit, and profit margin.
